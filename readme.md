How to running this code:

1.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    ```
    
2.  **Activate the virtual environment:**
    ```bash
    venv\Scripts\activate
    ```
    
3.  **Install Django:**
    ```bash
    pip install django
    ```
    
4.  **Run the development server:**
    ```bash
    python manage.py runserver
    ```
